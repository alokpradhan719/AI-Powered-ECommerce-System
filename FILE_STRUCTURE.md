## Data Flow

1. **User Input** → QueryBuilder.tsx
2. **API Request** → lib/api.ts
3. **Backend Processing** → server.js
4. **AI Processing** → OpenAI API
5. **Database Query** → MySQL
6. **Response** → ResultsDisplay.tsx
7. **Insights Display** → User interface
